--실행
1. Node.js가 안 깔려있다면 설치 (https://nodejs.org/ko/)
2. 설치한 폴더에 cmd를 열고 http-server가 안 깔려있다면 'npm i -g http-server' 입력
3. 위 모듈 설치가 끝나면 'npx http-server -c-1' 입력
4. 브라우저에서 'http://localhost:8080/' 접속
